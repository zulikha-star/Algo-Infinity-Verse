🦀 Why Learn Rust?

🔒

Memory Safety

Rust prevents null pointer dereferences, buffer overflows and data races at compile time — no garbage collector needed!

⚡

Performance

Rust is as fast as C and C++ with zero-cost abstractions. Used by Firefox, Discord, Cloudflare and AWS.

🔧

Concurrency

Fearless concurrency — Rust's ownership model prevents data races at compile time making concurrent code safe.

🌐

WebAssembly

Rust compiles to WebAssembly for high-performance web applications running at near-native speed in browsers.

📚 Core Rust Concepts

🔑 Ownership

Every value has a single owner. When owner goes out of scope, value is dropped automatically. 

let s1 = String::from("hello");
let s2 = s1; // s1 moved to s2
// println!("{}", s1); // Error!
println!("{}", s2); // OK

📎 Borrowing

References allow you to use a value without taking ownership. Use & for immutable, &mut for mutable. 

fn print_len(s: &String) {
    println!("Length: {}", s.len());
}
let name = String::from("Rust");
print_len(&name); // borrow
println!("{}", name); // still valid

🏗️ Structs

Custom data types that group related fields together. Similar to classes in other languages. 

struct User {
    name: String,
    age: u32,
}
let user = User {
    name: String::from("Alice"),
    age: 25,
};
println!("{}", user.name);

📋 Enums

Enums allow defining a type with multiple variants. Option and Result are built-in enums. 

enum Direction {
    North, South, East, West
}
let dir = Direction::North;
match dir {
    Direction::North => println!("Going North!"),
    _ => println!("Other direction"),
}

🎯 Pattern Matching

match is Rust's powerful control flow operator. It compares a value against patterns exhaustively. 

let number = 7;
match number {
    1 => println!("One"),
    2 | 3 => println!("Two or Three"),
    4..=6 => println!("Four to Six"),
    _ => println!("Something else"),
}

🔄 Concurrency

Rust's ownership prevents data races at compile time, making concurrent programming fearless and safe. 

use std::thread;
let handle = thread::spawn(|| {
    println!("Hello from thread!");
});
handle.join().unwrap();
println!("Main thread done!");
