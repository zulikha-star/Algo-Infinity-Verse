📊 Intent States Explained

🔴

Stuck

Detected when you have repeated errors with no meaningful code changes between runs.

Signals:
  • 3+ consecutive errored runs
  • No meaningful edits between runs
  • Same error repeating
💡 Hint:

Take a break, re-read the problem, or ask for a hint in our chatbot!

🟡

Guessing

Detected when you run code rapidly with minimal changes — suggesting trial and error.

Signals:
  • 3+ runs within 30 seconds
  • Less than 10 characters changed
  • No clear logical progression
💡 Hint:

Slow down! Plan your approach on paper before running code.

🟢

Confident

Detected when you have steady edits followed by successful runs — great progress!

Signals:
  • Meaningful edits between runs
  • Decreasing error rate
  • Successful run achieved
💡 Hint:

Great work! Consider optimizing your solution or trying a harder problem.

🟣

Overthinking

Detected when you make many small edits with long pauses but never run the code.

Signals:
  • 45+ second pause between runs
  • 5+ small edits with no run
  • Frequent minor changes
💡 Hint:

Trust your instincts! Run your code and see what happens.
