Hardware Bus: Idle

CPU Instruction Terminal

Execute Read and Write instructions across 4 independent CPU cores. The hardware System Bus uses the MESI Protocol to snoop traffic and automatically invalidate stale L1 cache lines.

MESI State Machine

M Modified: Dirty. I own it.
E Exclusive: Clean. Only I have it.
S Shared: Clean. Others have it.
I Invalid: Empty or Stale.
