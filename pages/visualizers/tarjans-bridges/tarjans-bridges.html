Theory

Tarjan's Algorithm finds "Critical Connections" (Bridges). A bridge is an edge whose removal disconnects the graph. It uses a single DFS pass to track discovery times and cycle closures.

  • Tree Edge Forward traversal building the DFS Spanning Tree.
  • Back Edge Connects a node to an already visited ancestor, proving a cycle exists (Not a bridge).
  • Bridge An edge where the child node has NO back-edge to an ancestor (`low[child] > disc[parent]`).

Complexity

Time Complexity O(V + E)
Space Complexity O(V)

Visits every vertex and traverses every edge exactly once. `disc` and `low` arrays take O(V) space.
