Theory

Finding the shortest path that visits all nodes in a cyclic graph requires BFS. However, a standard `visited` array causes infinite loops.

We solve this by memoizing the path history as an integer Bitmask.

  • Bitwise OR `mask | (1 << v)` flags node $v$ as visited in the binary string.
  • Cycle Pruning `dp[mask][v]` allows us to revisit node $v$, BUT ONLY IF doing so unlocks a new, previously unvisited node in the mask.

Complexity

Time Complexity O(N × 2^N)
Space Complexity O(N × 2^N)

The state space is bounded by the $2^N$ possible mask combinations multiplied by the $N$ possible current nodes.
