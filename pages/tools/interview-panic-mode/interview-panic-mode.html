Consistency
All nodes return same data at same time
Availability
Every request gets a response (not error)
Partition Tolerance
System works despite network partitions
Real systems must choose CP or AP — Partition Tolerance is non-negotiable in distributed systems.
CP examplesHBase, ZooKeeper, MongoDB (strong)
AP examplesCassandra, DynamoDB, CouchDB
Horizontal scalingAdd more machines (stateless services)
DB shardinghash(key) % N shards; avoid hotspots
